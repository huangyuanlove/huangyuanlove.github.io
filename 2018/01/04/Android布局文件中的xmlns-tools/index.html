AndroidеёғеұҖж–Үд»¶дёӯзҡ„xmlns:tools

еңЁдҪҝз”ЁAndroidStudioеҲӣе»әеёғеұҖж–Үд»¶зҡ„ж—¶еҖҷпјҢи·ҹеёғеұҖдёӢжҖ»жҳҜжңүеҰӮдёӢд»Јз Ғпјҡ

дәҺжҳҜеңЁе®ҳзҪ‘жҹҘдәҶдёҖдёӢиҝҷдҝ©иҙ§жҳҜе№Іеҳӣз”Ёзҡ„пјҢдёӢйқўжҳҜиҮӘе·ұзҡ„зҝ»иҜ‘+е®һи·ө

xmlnsзҡ„е…Ёз§°жҳҜxmlnamespace,е’Ңc++дёӯnamespaceе·®дёҚеӨҡпјҢйғҪжҳҜдёәдәҶи§ЈеҶіе‘ҪеҗҚдёҠзҡ„еҶІзӘҒй—®йўҳгҖӮ
еңЁAndroidеёғеұҖж–Үд»¶дёӯпјҢеёёи§Ғзҡ„xmlnsеӨ§жҰӮжңүдёүдёӘпјҲдҪ еҸҜд»ҘйҡҸж„Ҹе‘ҪеҗҚпјҢжҠҠtoolsж”№жҲҗbugд№ҹеҸҜд»ҘпјҢеҸӘиҰҒеҜ№еә”дҪҝз”Ёtoolsзҡ„ең°ж–№йғҪж”№жҲҗbugпјҢиҜҙзҷҪдәҶпјҢе®ғеҸӘжҳҜдёӘеҸҳйҮҸиҖҢе·ІпјүгҖӮ

android

з”ЁдәҺAndroidзі»з»ҹе®ҡд№үзҡ„дёҖдәӣеұһжҖ§

app

з”ЁдәҺжҲ‘д»¬иҮӘе®ҡд№үзҡ„дёҖдәӣеұһжҖ§

tools

з»ҷIDEжҲ–иҖ…йў„и§Ҳз•Ңйқўз”Ёзҡ„пјҢеҪ“жү“еҢ…зј–иҜ‘ж—¶е№¶дёҚдјҡеҢ…еҗ«еңЁapkдёӯгҖӮ

toolsеҸҜд»Ҙе№Ід»Җд№Ҳ

е…ҲзңӢе®ҳзҪ‘д»Ӣз»Қ https://developer.android.com/studio/write/tool-attributes.html

Android Studio supports a variety of XML attributes in the tools namespace that enable design-time features (such as which layout to show in a fragment) or compile-time behaviors (such as which shrinking mode to apply to your XML resources). When you build your app, the build tools remove these attributes so there is no effect on your APK size or runtime behavior.

еӨ§иҮҙж„ҸжҖқе°ұжҳҜиҜҙдҪҝз”ЁtoolsеҗҺйқўзҡ„еұһжҖ§дёҚдјҡеҶҚзј–иҜ‘ж—¶еӯҳеңЁпјҢеҸӘеӯҳеңЁдәҺи®ҫи®Ўж—¶зҡ„йў„и§ҲпјҢдёҚдјҡеҪұе“Қapkзҡ„дҪ“з§ҜпјҢ

Error handling attributes

еҪұе“ҚLintжҸҗзӨәзҡ„еұһжҖ§дё»иҰҒжңүдёӢйқўдёүз§Қпјҡ

ignoreеұһжҖ§жҳҜе‘ҠиҜүLintеҝҪз•Ҙxmlдёӯзҡ„жҹҗдәӣиӯҰе‘ҠпјҢжҜ”еҰӮжҲ‘д»¬еңЁImageViewжҲ–иҖ…ImageButtonдёӯжІЎжңүеҶҷcontentDescriptionеұһжҖ§пјҢLintдјҡжңүиӯҰе‘ҠпјҡMissing contentDescription attribute on image,еҺҹеӣ жҳҜиҝҷдёӘеұһжҖ§жҳҜжҸҗдҫӣж— йҡңзўҚйҳ…иҜ»зҡ„пјҢжІЎжңүиҝҷдёӘеұһжҖ§зҡ„иҜқScreen Readerж— жі•жӯЈеёёе·ҘдҪңпјҢдҪҶжҳҜжңүдәӣзү№е®ҡеҲҶзұ»зҡ„иҪҜд»¶жҳҜдёҚиҖғиҷ‘иҝҷдәӣдёңиҘҝзҡ„гҖӮиҷҪз„¶иҝҷз§ҚиӯҰе‘Ҡж— жүҖи°“пјҢдҪҶжҳҜеҜ№дәҺиҰҒжұӮд»Јз ҒдёӯдёҚиғҪжңүwarningзҡ„е…¬еҸёжқҘиҜҙпјҢиҝҷжҳҜдёҚеҸҜеҸ–зҡ„пјҢжҲ‘д»¬еҸҜд»Ҙз”ЁignoreеұһжҖ§жқҘеҝҪз•ҘиҝҷдёӘиӯҰе‘Ҡ(еҰӮжһңдҪ иҜҙдҪ ж”№дәҶLintзҡ„иӯҰе‘Ҡзә§еҲ«пјҢеҪ“жҲ‘жІЎиҜҙ)пјҡ

targetApiе’Ңд»Јз Ғдёӯзҡ„жіЁи§Ј@RequiresApi(api = Build.VERSION_CODES.JELLY_BEAN_MR2)е·®дёҚеӨҡгҖӮеҒҮи®ҫй…ҚзҪ®ж–Үд»¶дёӯзҡ„жңҖе°ҸsdkLevelдёә11пјҢиҖҢеёғеұҖж–Үд»¶дёӯдҪҝз”ЁдәҶ21зҡ„жҺ§д»¶жҜ”еҰӮGridLayout,LintдјҡжңүиӯҰе‘ҠпјҢдёәдәҶж¶ҲйҷӨиҝҷдёӘиӯҰе‘ҠпјҢеҸҜд»Ҙиҝҷд№ҲеҶҷпјҡ

localeжҳҜе‘ҠиҜүLintе’ҢAndroidStudio editorй»ҳи®Өзҡ„жҳҜд»Җд№ҲиҜӯиЁҖпјҢеҰӮжһңдёҚжҢҮе®ҡзҡ„иҜқпјҢй»ҳи®ӨжҳҜиӢұиҜӯгҖӮеҸҜд»ҘжҠҠиҝҷдёӘеұһжҖ§ж·»еҠ еҲ°values/strings.xmlдёӯпјҡ

tools: instead of android

Intended for: View
Used by: Android Studio layout editor

жҲ‘д»¬жңүдёҖдёӘTextViewпјҢйңҖиҰҒжҳҫзӨәд»ҺзҪ‘з»ңиҺ·еҸ–еҲ°зҡ„ж–Үд»¶пјҢжҺ§д»¶еӨ§е°ҸжҳҜwrap_content,дҪҶжҳҜTextViewдёӯжІЎжңүж–Үеӯ—зҡ„иҜқеңЁйў„и§Ҳз•ҢйқўдёӯжҳҜзңӢдёҚеҲ°зҡ„пјҢеӨ§йғЁеҲҶеҗҢеӯҰеҸҜиғҪдјҡдҪҝз”Ёandroid:text=XXXXиҝҷдёӘеұһжҖ§пјҢи°ғж•ҙеҘҪеёғеұҖд№ӢеҗҺеҶҚжҠҠж–Үеӯ—еҲ йҷӨгҖӮдёҖдёӨдёӘжҺ§д»¶иҝҳеҘҪиҜҙпјҢжҺ§д»¶еӨҡдәҶжҢҮдёҚе®ҡе“ӘдёӘжҺ§д»¶е°ұеҝҳи®°еҲ йҷӨж–Үжң¬дәҶпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”ЁtoolsиҝҷдёӘдёңиҘҝпјҡ

иҝҷж—¶жҲ‘д»¬еңЁIDEйў„и§Ҳз•ҢйқўжҳҜеҸҜд»ҘзңӢеҲ°ж–Үеӯ—зҡ„пјҢжү“еҢ…жҲҗapkиҝҗиЎҢеңЁжүӢжңәдёҠж—¶жҳҜзңӢдёҚеҲ°зҡ„гҖӮ
жҖ»д№ӢпјҢtoolsеҸҜд»Ҙе‘ҠиҜүAndroid Studioе“ӘдәӣеұһжҖ§еңЁиҝҗиЎҢзҡ„ж—¶еҖҷжҳҜиў«еҝҪз•Ҙзҡ„пјҢеҸӘеңЁи®ҫи®ЎеёғеұҖзҡ„ж—¶еҖҷжңүж•ҲгҖӮеҹәжң¬дёҠеҺҹз”ҹжҺ§д»¶зҡ„еұһжҖ§йғҪеҸҜд»Ҙиҝҷд№ҲдҪҝз”ЁгҖӮ
